MPEG_FILTER ( VÍDEO, FILTER )

Permite escoger el tipo de filtro utilizado para renderizar un vídeo

PARÁMETROS
INTVÍDEO : Identificador del vídeo
INTFILTER : Tipo de filtero
DESCRIPCIÓN

Permite escoger uno de los filtros disponibles para reproducir un vídeo concreto, entre los siguientes:

  • 0: Sin filtrar. El vídeo se reproduce tal cual, sin aplicar ningún filtro al resultado.
  • 1: Filtro bilinear. Este filtro "suaviza" la imagen mezclando pixels adyacentes. Es un filtro adecuado para vídeos en baja resolución.
  • 2: Deblocking. Este filtro suaviza ciertos defectos en la reproducción de vídeos, sin modificar apenas el resto de la imagen.

Por defecto, el filtro activado para cualquier vídeo es el 0 (o sea, ninguno).

VEA TAMBIÉN
LOAD_MPEG Recupera de disco un fichero de vídeo
PLAY_MPEG Inicia la reproducción de un vídeo
