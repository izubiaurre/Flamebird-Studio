MAP_CLEAR ( LIBRERÍA, GRÁFICO, COLOR )

Borra el contenido de un gráfico

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico dentro de la librería
INTCOLOR : Número de color
DESCRIPCIÓN

Borra el contenido de un gráfico, estableciendo todos los pixels al color indicado.

El significado del parámetro color depende de la profundidad de color del gráfico. Para un gráfico de 8 bits, representa un número de color de 0 a 255. Para un gráfico de 16 bits, es un número de 0 a 65535 con un código de color que varía en función del modo gráfico. Puedes utilizar la función RGB para obtener la codificación de un color determinado a partir de sus componentes.

En cualquier caso, el color 0 representa un pixel transparente, tanto en 8 como en 16 bits.

VEA TAMBIÉN
