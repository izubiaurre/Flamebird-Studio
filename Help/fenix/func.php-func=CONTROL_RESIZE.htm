CONTROL_RESIZE ( CONTROL, ANCHO, ALTO )

Cambia el tamaño de un control

PARÁMETROS
INTCONTROL : Identificador de un control
INTANCHO : Nuevo ancho del control, en pixels
INTALTO : Nuevo alto del control, en pixels
DESCRIPCIÓN

Cambia el tamaño de un control en una ventana. Es posible redimensionar incluso un control que no sea redimensionable de por sí, aunque en la mayoría de los casos eso no permitirá obtener el resultado deseado. Sin embargo, no es posible ampliar o reducir un control más allá de los límites de tamaño mínimo o máximo que él mismo imponga.

Esta función no desplaza otros controles por la ventana ni hace ningún otro tipo de operación sofisticada, por lo que es muy probable que se acabe con controles que se superpongan si no se tiene cuidado. Además, al contrario que la función WINDOW_ADDCONTROL , esta función no ajusta el tamaño de la ventana para que quepa el control redimensionado.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
