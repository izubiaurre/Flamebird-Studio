SAVE_FNT ( FUENTE, "FICHERO" )

Graba en disco un tipo de letra FNT

PARÁMETROS
INTFUENTE : Número de tipo de letra
STRINGFICHERO : Nombre de fichero
DESCRIPCIÓN

Crea o sobreescribe un fichero en disco, con el nombre especificado, que contenga un tipo de letra actualmente ubicado en memoria. Este tipo de letra habrá sido recuperado de disco previamente por la función LOAD_FNT u otra equivalente. Se guardará cualquier cambio efectuado en memoria al tipo de letra con funciones como SET_GLYPH . Además, el fichero se guardará en formato comprimido.

VEA TAMBIÉN
LOAD_FNT Recupera de disco un tipo de letra FNT
GET_GLYPH Recupera el gráfico de un carácter
SET_GLYPH Recupera el gráfico de un carácter
