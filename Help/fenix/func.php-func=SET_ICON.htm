SET_ICON ( LIBRERÍA, GRÁFICO )

Usa un gráfico como icono para la ventana del juego

PARÁMETROS
INTLIBRERÍA : Librería que contiene el gráfico
INTGRÁFICO : Código del gráfico
DESCRIPCIÓN

Esta función permite escoger el icono de la ventana del juego, cuando el juego no funciona a pantalla completa. Es necesario llamar a esta función antes que llamara a SET_MODE , ya que esa es la función que creará y visualizará dicha ventana. Recuerda que SET_MODE inicializa en un principio el juego a pantalla completa: para inicializar el modo de ventana, es preciso cambiar el valor de la variable FULL_SCREEN antes de llamar a SET_MODE .

Para esta función, debes utilizar un gráfico de 32x32 pixels. El gráfico puede estar a 8 ó 16 bits de color, y no debes nunca borrar este gráfico de memoria usando funciones como UNLOAD_MAP o UNLOAD_FPG .

No es posible cambiar el icono de una ventana ya inicializada, pero puedes volver a llamar a SET_MODE para activar los cambios.

VEA TAMBIÉN
FULL_SCREEN 
SET_MODE Establece un modo gráfico o crea la ventana del juego
SET_TITLE Establece el título de la ventana del juego
