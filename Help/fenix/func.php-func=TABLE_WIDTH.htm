TABLE_WIDTH ( TABLA, [ "NOMBRE" ], [ NOMBRE ] )

Permite obtener o cambiar el ancho de una columna, o de toda una tabla

PARÁMETROS
INTTABLA : Identificador devuelto por TABLE
STRING NOMBRE : (opcional)   Nombre de la columna
INT NOMBRE : (opcional)   Nuevo ancho
VALOR DE RETORNO

INT   :   Ancho de la columna

DESCRIPCIÓN

Esta función dispone de tres formas diferentes de uso.

  • Si es llamada con un sólo parámetro, devuelve el ancho en pixels que ocupan todas las columnas de la misma.
  • Si se le pasa como segundo parámetro una columna de una tabla (por su nombre, u opcionalmente por su número de índice, con 0 para la columna más a la izquierda), devuelve sólo el ancho en pixels de dicha columna.
  • Además, si se le pasa un tercer parámetro opcional, cambiará el tamaño de dicha columna al nuevo tamaño especificado por el usuario en este parámetro.
