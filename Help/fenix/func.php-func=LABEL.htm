INT LABEL ( "TEXTO", [ ALINEACIÓN ], [ ANCHO ] )

Crea una etiqueta de texto estático

PARÁMETROS
STRINGTEXTO : Texto de la etiqueta
INT ALINEACIÓN : (opcional)   Alineación de la etiqueta
INT ANCHO : (opcional)   Ancho del control en pixels
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea una etiqueta de texto. La etiqueta puede tener varias líneas, separadas por el caracter de salto de línea CHR(10), y opcionalmente alguna letra subrayada si se precede del carácter & . Cuando el usuario pulse ALT junto con la tecla subrayada de una de las etiquetas dentro de la ventana actual, el sistema moverá el foco de entrada al siguiente control que fuese añadido a la ventana justo después de la etiqueta.

Es posible especificar un alineamiento a la izquierda (0), al centro (1) o a la derecha (2). Esta opción sólo tiene sentido en etiquetas de múltiples líneas, ya que el ancho del control se ajusta automáticamente a su contenido.

Es posible especificar un ancho en pixels para el control (por defecto se ajusta al ancho mínimo de la etiqueta). En este caso, el texto se alineará sobre el ancho escogido, y el contról se marcará como redimensionable horizontalmente.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
LABEL_TEXT Cambia el texto de una etiqueta estática
