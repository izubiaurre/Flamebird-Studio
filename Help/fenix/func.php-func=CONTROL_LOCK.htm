CONTROL_LOCK ( CONTROL, HORIZONTAL, VERTICAL )

Bloquea el tamaño de un control, impidiendo que cambie

PARÁMETROS
INTCONTROL : Identificador de un control
INTHORIZONTAL : 1 para bloque el control horizontalmente
INTVERTICAL : 1 para bloque el control verticalmente
DESCRIPCIÓN

Bloque el tamaño de un control, impidiendo que sea modificado por el usuario respecto a su original. Es útil cuando en un cuadro de diálogo complejo interesa que sólo algunos controles sean modificados por una operación de cambio de tamaño de la ventana. Una vez bloqueado, un control no puede desbloquearse de nuevo. Es posible bloquear el control sólo en la dirección vertical u horizontal, o ambas, en función de cuales de los dos últimos parámetros de esta función contienen 1 (TRUE).

Se recomienda usar esta función antes de visualizar la ventana donde se ubique el control.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
