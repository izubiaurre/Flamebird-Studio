XPUT ( LIBRERÍA, GRÁFICO, X, Y, ÁNGULO, TAMAÑO, FLAGS, REGION )

Dibuja un gráfico en el fondo de pantalla

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico a dibujar
INTX : Coordenada horizontal
INTY : Coordenada vertical
INTÁNGULO : Ángulo en miligrados (90000 = 90º)
INTTAMAÑO : Tamaño en porcentaje (100 = tamaño original)
INTFLAGS : Flags para el dibujo
INTREGION : Región de recorte (0 = ninguna)
DESCRIPCIÓN

Esta función dibuja un gráfico en el fondo de pantalla, en las coordenadas especificadas como parámetro, y aplicando todas las posibilidades de dibujo de un proceso (ángulo de rotación, transparencias, escalado...). El punto especificado equivaldrá al lugar donde se dibuje el centro del gráfico, y a partir de cual se rotará o cambiará de tamaño.

Esta función es útil especialmente para "preparar" pantallas o niveles a partir de bloques gráficos.

VEA TAMBIÉN
XPUT Dibuja un gráfico en el fondo de pantalla
MAP_PUT Dibuja un gráfico dentro de otro
