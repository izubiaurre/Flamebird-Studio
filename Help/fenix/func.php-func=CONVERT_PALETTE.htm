CONVERT_PALETTE ( LIBRERÍA, GRÁFICO, &TABLA )

Modifica los valores de pixel de un gráfico según una tabla

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico dentro de la librería
POINTERTABLA : Tabla de 256 bytes con los nuevos valores de pixel
DESCRIPCIÓN

Esta función permite hacer conversiones de color en un gráfico de 8 bits. A diferencia del resto de funciones de paleta de 8 bits, alterará efectivamente los valores de pixel del gráfico.

Para utilizarla es preciso crear una tabla de 256 valores de tipo byte, de manera que cada valor contendrá el nuevo valor para cada uno de los 256 pixels diferentes posibles. El primer byte contendrá el nuevo valor que se asignará a los pixels que estén a 0, el segundo byte a los que estén a 1, y así sucesivamente.

A veces es deseable crear gráficos con zonas que sea posible cambiar de color fácilmente para obtener varias versiones del gráfico con pequeñas diferencias, como los colores de las camisetas de los sprites de un juego deportivo o el color de piel de un monstruo. Para ello, sería preciso determinar una serie de colores en el gráfico que interesaría cambiar por otros. Esta función permite lograr ese efecto fácilmente: basta con preparar un array de bytes que contenga inicialmente todos los números del 0 a 255 ordenados de menor a mayor, y modificar en su interior los colores que desean cambiarse por los nuevos valores.

Hay que tener en cuenta que el cambio es de una sóla dirección: al cambiar los valores de pixel de cada gráfico, es preciso mantener en memoria varias copias de cada gráfico del que se desee crear varias versiones empleando esta función. Para ello es posible emplear la función MAP_CLONE .

VEA TAMBIÉN
MAP_CLONE Crea una copia de un gráfico
SET_COLORS Cambia inmediatamente los valores de la paleta de colores
ROLL_PALETTE Rota una sección de colores de la paleta
