DESCRIPCIÓN

Esta sentencia equivale a una cadena de IF/ELSEIF donde una expresión se evalúa y, en función del resultado, se ejecuta un bloque de código u otro.

En el interior del bloque switch habrá uno o más bloques CASE y opcionalmente un bloque DEFAULT. Cada bloque CASE contiene una lista de expresiones separadas por comas que corresponden a valores que puede tomar la expresión que aparece entre paréntesis a la derecha del SWITCH. Si el resultado de evaluar dicha expresión es uno de los valores proporcionados por el CASE, se ejecutará su bloque de código.

Las expresiones a la derecha de un CASE también pueden incluir rangos usando el operador especial "..". Por ejemplo, CASE 0..9 aceptaría todos los números entre 0 y 9 inclusives.

El bloque DEFAULT se ejecutará sólo si la expresión devuelve un valor que no está contemplado en ninguno de los CASE dentro del SWITCH.

Tipos de dato en SWITCH

SWITCH acepta sólo cadenas (tanto STRING como cadenas de ancho fijo en un array o puntero a CHAR) o enteros. Cualquier otro tipo de dato se intentará convertir a entero o dará un error de compilación.
