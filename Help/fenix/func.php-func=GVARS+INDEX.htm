Variables globales

Las variables globales son accesibles desde cualquier proceso en Fenix. Sin embargo, por cada variable, sólo hay una copia. En esto se diferencias de las variables locales o privadas, de las cuales existe una instancia de la variable para cada proceso existente.

Las variables globales predefinidas suelen almacenar datos de configuración general o datos informativos sobre el estado del programa, como FPS .

CONTENIDO
ALPHA_STEPS Niveles de transparencia necesitados
ARGC Contiene el número de parámetros recibido por el programa
ARGV Contiene los parámetros recibido por el programa
ASCII Contiene información del último carácter que se ha pulsado en el teclado
FADING Estado de la operación de fundido
FPS Indica la velocidad actual en Frames/Segundo del programa
SCAN_CODE Contiene el código de tecla pulsado en el último frame
SHIFT_STATUS Almacena el estado de las teclas de control
TEXT_Z Profundidad de escritura de los textos
TIMER Diez cronómetros programables accesibles en todo momento
EXIT_STATUS Se pone a 1 cuando el usuario trata de cerrar la aplicación
FOCUS_STATUS Se pone a 1 cuando la aplicación está en primer plano
MOUSE_STATUS Se pone a 1 cuando el ratón está sobre la ventana
WINDOW_STATUS Se pone a 0 cuando la ventana está minimizada
