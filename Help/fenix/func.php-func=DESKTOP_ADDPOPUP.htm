DESKTOP_ADDPOPUP ( VENTANA, CONTROL, X, Y )

Muestra un control "popup"

PARÁMETROS
INTVENTANA : Identificador de la ventana origen
INTCONTROL : Identificador del control a añadir
INTX : Coordenada horizontal
INTY : Coordenada vertical
DESCRIPCIÓN

Esta función muestra un control "popup" en las coordenadas especificadas. Un control popup es un control especial que se muestra por encima de todas las ventanas existentes. Su principal utilidad consiste en crear menús de opciones o pequeños cuadros de aviso o "tooltips". El control popup será destruído automáticamente cuando el usuario haga clic en alguna ventana.

Es posible crear un control popup orientado a partir de una ventana. En ese caso, el primer parámetro deberá ser el identificador de una ventana existente, y las coordenadas se darán relativas a la esquina superior izquierda de la ventana, como en el caso de un control interno a la misma. Si se usa como ventana simplemente el número 0, el control popup se considera general, y las coordenadas serán un punto en pantalla.

La utilidad más común de este control es crear un menú pop-up mediante la función POPUP_MENU y mostrarlo en la posición actual del ratón (ventana 0 y coordenadas MOUSE.X y MOUSE.Y).

Es importante recordar que el control será destruído automáticamente, es decir, no es válido emplear el identificador de un control que ya haya sido utilizado en una ventana o en un popup mostrado anteriormente. Hay que crear un nuevo control cada vez que se desee mostrar un popup nuevo.

VEA TAMBIÉN
MENU Crea un nuevo menú
DESKTOP_REMOVEPOPUP Quita de pantalla un control "popup" y lo destruye
