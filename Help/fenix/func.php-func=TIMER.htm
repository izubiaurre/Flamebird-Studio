TIMER

Diez cronómetros programables accesibles en todo momento

DESCRIPCIÓN

Fenix dispone de 10 temporizadores accesibles a través del array de variables timer (desde timer[0] a timer[9]). Como ocurre con cualquier array acceder a timer como una variable entera es lo mismo que acceder a timer[0].

Estos temporizadores se inicializan automáticamente al comenzar el programa al valor 0 y se incrementan automáticamente 100 veces por segundo.

En cualquier momento puede cambiarse el valor de uno de ellos, por ejemplo inicializándolo de nuevo a 0, para controlar una cantidad de tiempo concreta. Por ejemplo, para realizar un reloj se debería comprobar el timer elegido contra un valor mayor a 100 y restarle 100 a ese temporizador.

NOTAS
Las variables globales se actualizan cada instrucción FRAME a efectos de lenguaje por tanto los valores leidos en un temporizador no serán consecutivos a menos que nuestro programa se esté ejecutando a 100 frames por segundo.
Las comprobaciones con los temporizadores deben realizarse siempre con los operadores de > = o > y nunca con el = ya que no se puede asegurar que tras la actualización del temporizador el valor sea exactamente igual al deseado.
