GUI_POINTER ( [ PUNTERO ] )

Modifica u obtiene el puntero del ratón del GUI

PARÁMETROS
INT PUNTERO : (opcional)   Nuevo valor del puntero
VALOR DE RETORNO

INT   :   Código del puntero de ratón actual

DESCRIPCIÓN

Esta DLL proporciona una serie de punteros de ratón estándares. Esta función permite escoger el puntero por defecto, que aparecerá siempre en lugar de la flecha blanca. Si se especifica, el nuevo código puede ser cualquiera de los siguientes (constantes definidas en GUI.INC):

  • POINTER_ARROW
  • POINTER_RESIZEH
  • POINTER_RESIZEV
  • POINTER_RESIZENW
  • POINTER_RESIZENE
  • POINTER_CROSS
  • POINTER_IBEAM
  • POINTER_RESIZEH2
  • POINTER_HAND
  • POINTER_DRAGOBJECT
  • POINTER_ARROWCOPY
  • POINTER_BADDROP
