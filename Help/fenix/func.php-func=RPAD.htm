STRING RPAD ( "CADENA", TAMAÑO )

Expande una cadena añadiendo espacios a la derecha

PARÁMETROS
STRINGCADENA : Cadena original
INTTAMAÑO : Tamaño deseado
VALOR DE RETORNO

STRING   :   Cadena resultante

DESCRIPCIÓN

Esta función devuelve una cadena creada tras añadir espacios (carácter 32: " ") al final de la cadena original, hasta conseguir que alcance una longitud igual a la especificada como segundo parámetro.

Si la cadena original tiene un tamaño igual o superior al especificado, esta función devuelve la cadena original sin ningún cambio. Si se desea forzar el límite de tamaño, truncando la cadena original, puede usarse la función SUBSTR .

VEA TAMBIÉN
LPAD Expande una cadena añadiendo espacios a la izquierda
