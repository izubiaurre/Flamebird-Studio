EXIT ( [ TEXTO ], [ SALTO ] )

Sale del juego inmediatamente

PARÁMETROS
INT TEXTO : (opcional)   Texto a mostrar al usuario
INT SALTO : (opcional)   Código de retorno de la aplicación
DESCRIPCIÓN

Esta función abandona la ejecución del programa inmediatamente. El parámetro texto permite especificar un texto libre a mostrar al usuario que podría visualizarse en pantalla dependiendo de la plataforma (no se garantiza). El código de retorno de la aplicación tiene utilidad en algunos sistemas operativos, y es un número entero que puede comprobarse por los programas que ejecutan otros programas.

Ambos parámetros son opcionales. Si no se especifican, no se mostrará ningún mensaje, y el código de salida del programa será 0.
