TABLE_CHANGE ( TABLA, FILA, "NOMBRE", "VALOR" )

Devuelve el valor de una celda en una tabla

PARÁMETROS
INTTABLA : Identificador devuelto por TABLE
INTFILA : Número de fila (0 para la primera)
STRINGNOMBRE : Nombre de la columna
STRINGVALOR : Nuevo valor a asignar
DESCRIPCIÓN

Dada una tabla, un número de fila, y un nombre o índice de columna, esta función permite modificar el contenido de la celda correspondiente, cambiándolo por la cadena que se le pase como parámetro. Si la columna en cuestión es una columna de gráficos, este valor se interpreta como número entero, e identifica el gráfico en cuestión.

VEA TAMBIÉN
TABLE Crea una tabla o lista
TABLE_ADDCOLUMN Añade una columna a una tabla, opcionalmente con datos
TABLE_ADDCOLUMN_BITMAPS Añade una columna a una tabla, opcionalmente con datos
