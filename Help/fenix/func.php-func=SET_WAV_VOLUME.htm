SET_WAV_VOLUME ( CANAL, VOLUMEN )

Especifica el nuevo volumen de sonido para el efecto indicado

PARÁMETROS
INTCANAL : Identificador del efecto devuelto por LOAD_WAV
INTVOLUMEN : Nuevo volumen, de 0 a 128
DESCRIPCIÓN

Cambia el volumen de un efecto de sonido cargado anteriormente mediante LOAD_WAV , entre 0 (silencio) y 128 (volumen original del efecto).

El volumen final de reproducción del efecto será este modificado por el valor de volumen de reproducción asignadoi al canal por el que esté.

VEA TAMBIÉN
SET_CHANNEL_VOLUME Especifica el volumen general del canal de sonido
PLAY_WAV Reproduce un efecto de sonido
STOP_WAV Detiene la reproducción de un efecto de sonido
PAUSE_WAV Detiene temporalmente la reproducción de un efecto de sonido
