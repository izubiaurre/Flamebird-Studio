Provoca que una sección de código se compile sólo si una macro no está definida

PARÁMETROS
INTCONSTANTE : Nombre de una macro #define
DESCRIPCIÓN

Esta directiva del preprocesador permite marcar una sección de código (todas las líneas entre #ifdef y #endif) como opcional. El compilador sólo recibirá estas líneas siempre y cuando la constante cuyo nombre se especifica tras #ifdef no se encuentra definida (una definición en blanco se considera una definición válida a todos los efectos).

Es importante entender que el compilador no recibirá las líneas excluídas, por lo que no se detectarán errores de sintaxis o de ningún otro tipo en dichas líneas.

VEA TAMBIÉN
#IFDEF Provoca que una sección de código se compile sólo si una macro está definida
#DEFINE Asigna un texto a una macro
