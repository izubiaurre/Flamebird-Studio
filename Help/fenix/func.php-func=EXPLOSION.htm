EXPLOSION ( NUBES, TAMAÑO, FRAMES, BLUR, DENSIDAD, WIDTH, HEIGHT )

Genera una explosión con los parámetros indicados

PARÁMETROS
INTNUBES : Numero de nubes de partículas que se crearán.
INTTAMAÑO : Tamaño máximo que alcanzará la nube. En pixels.
INTFRAMES : Número de Frames que generará la explosión. (1-20)
INTBLUR : Numero de veces que se difuminará el gráfico (1-5).
INTDENSIDAD : Densidad de las partículas en una nube.
INTWIDTH : Anchura del mapa que se generará.
INTHEIGHT : Altura del mapa que se generará.
DESCRIPCIÓN

Crea una explosión para posteriormente reproducirla mediante MOVE_EXPLOSION .

Esta función devuelve un mapa de 8 bits donde se generará la explosión. Puede usarse esta función tanto en 8 como en 16 bits , pero el mapa devuelto será siempre de 8 y suará la paleta que tengamos guardada en memoria. En caso de que una explosión se haye en proceso o no se pueda crear una nueva devolverá un -1.

VEA TAMBIÉN
MOVE_EXPLOSION Avanza un frame una explosión
