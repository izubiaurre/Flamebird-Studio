TABLE_SET ( TABLA, TIPO, VALOR )

Modifica parámetros y opciones de una tabla

PARÁMETROS
INTTABLA : Identificador devuelto por TABLE
INTTIPO : Tipo de parámetro a modificar
INTVALOR : Nuevo valor del parámetro
DESCRIPCIÓN

Modifica ciertos parámetros de comportamiento o aspecto de un control tipo tabla. El segundo parámetro puede ser uno de los siguientes valores definidos en GUI.INC (todos ellos salvo TABLE_CAPTION admiten únicamente como valor 0 ó 1):

  • TABLE_CAPTION: permite establecer el alto, en pixels, de la línea de título de la tabla. Un valor de 0 elimina la línea de título. Por defecto las tablas tienen título.
  • TABLE_RESIZABLE_COLUMNS: Si esta opción vale 1, el usuario podrá cambiar de tamaño las columnas de la tabla con la ayuda del ratón, lo cual requiere que la barra de título esté visible. Por defecto está a 1.
  • TABLE_SORTABLE_COLUMNS: Si esta opción vale 1, el usuario podrá ordenar la tabla haciendo clic sobre el título de una columna. Por defecto está activada.
  • TABLE_MOVEABLE_COLUMNS: Si esta opción vale 1, el usuario podrá desplazar las columnas a izquierda o derecha con el ratón, cambiando el orden en el que se muestra. Activada por defecto.
