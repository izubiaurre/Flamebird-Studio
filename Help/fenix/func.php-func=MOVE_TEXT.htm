MOVE_TEXT ( TEXTO, X, Y )

Mueve un texto dibujado mediante WRITE

PARÁMETROS
INTTEXTO : Identificador del texto o 0 para todos
INTX : Nueva coordenada horizontal
INTY : Nueva coordenada vertical
DESCRIPCIÓN

Esta función cambia la posición en pantalla de un texto dibujado mediante WRITE u otra función equivalente. El significado exacto de las nuevas coordenadas depende de la alineación del texto, tal como se dibujó originalmente. Esta función es útil para hacer efectos tales como un scroll vertical de créditos o similar.

VEA TAMBIÉN
DELETE_TEXT Elimina un texto dibujado mediante WRITE
WRITE Dibuja un texto en pantalla
