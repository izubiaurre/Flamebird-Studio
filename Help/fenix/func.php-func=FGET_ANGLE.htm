FGET_ANGLE ( X1, Y1, X2, Y2 )

Calcula el ángulo formado por la línea entre dos puntos

PARÁMETROS
INTX1 : Coordenada X del primer punto
INTY1 : Coordenada Y del primer punto
INTX2 : Coordenada X del segundo punto
INTY2 : Coordenada Y del segundo punto
DESCRIPCIÓN

Esta función devuelve, en milésimas de grado, el ángulo formado por la línea entre los dos puntos especificados, y el eje X (horizontal) de coordenadas.

Con esta función puede, fácilmente, orientar el ángulo de un proceso para que éste siempre esté "mirando" hacia un lugar concreto (como puede ser otro proceso). La aplicación más sencilla puede ser un proceso que "persiga" a otro simplemente utilizando FGET_ANGLE seguida de ADVANCE.

VEA TAMBIÉN
GET_ANGLE Calcula el ángulo de la línea entre dos procesos
ADVANCE Desplaza el proceso actual según su ángulo
