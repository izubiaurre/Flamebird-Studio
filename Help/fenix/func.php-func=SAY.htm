SAY ( "MESSAGE" )

Muestra un mensaje en la consola de DEBUG

PARÁMETROS
STRINGMESSAGE : Mensaje a mostrar en la consola
DESCRIPCIÓN

Esta función muestra en la consola (ALT+C en modo de compilación para DEBUG) una línea con el texto indicado. En ciertas versiones de Fenix, la línea se muestra también en la consola del sistema donde fuese ejecutado el intérprete.

La principal utilidad de esta función es escribir "chivatos", pequeñas inserciones de código que muestran en momentos clave el estado de variables importantes, o simplemente informan acerca de la ejecución de un trozo de código. Esto permite obtener información útil sobre el desarrollo del programa, de cara a detectar la causa de bugs molestos.
