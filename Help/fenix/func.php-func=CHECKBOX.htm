INT CHECKBOX ( "TEXTO", &VARIABLE )

Crea una caja de chequeo

PARÁMETROS
STRINGTEXTO : Texto de la caja de chequeo
POINTERVARIABLE : Puntero a la variable INT
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea una caja de chequeo, un control con dos estados: activado y desactivado, que el usuario puede cambiar y que se basan en el contenido de una variable INT, cuya dirección se le pasará a esta función como segundo parámetro empleando el operador & . Si esta variable contiene un valor CIERTO, la caja de chequeo estará marcada. El control cambiará el contenido de la variable automáticamente cuando el usuario manipule el control.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
