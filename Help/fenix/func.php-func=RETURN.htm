RETURN [ valor ];

Sentencia que finaliza la ejecución de una función o proceso

PARÁMETROS
INTvalor : Valor de retorno del proceso, del tipo especificado en la cabecera del proceso o función (INT por defecto)
DESCRIPCIÓN

Esta sentencia provoca la terminación inmediata del proceso actual y la muerte del mismo. Si el proceso actual es un FUNCTION , o un proceso que no ha ejecutado ninguna sentencia FRAME, el proceso que lo llamó puede recoger el valor de retorno opcional especificado.
