LOAD_MAP ( "FICHERO" )

Recupera de disco un gráfico MAP

PARÁMETROS
STRINGFICHERO : Nombre del fichero
DESCRIPCIÓN

Esta función recupera un gráfico MAP desde disco. El MAP es un formato propio de Fenix que permite almacenar gráficos de 1, 8 o 16 bits por pixel en formato comprimido, con pixels transparentes, y opcionalmente con una animación. Además, los gráficos de 8 bits incluyen la paleta de colores en el propio fichero MAP.

LOAD_MAP devuelve un número entero que identifica al gráfico MAP. Es posible utilizar a partir de entonces este valor con cualquier función que admita un gráfico como parámetro, indicando el código 0 de librería, o asignar el número a la variable GRAPH .

NOTAS
Sólo es posible visualizar gráficos de 16 bits si el modo gráfico actual es de 16 bits. En cambio, es posible visualizar gráficos de 8 bits en el modo de 16 bits.
Sólo hay una paleta de colores activa en todo momento. Usa sólo gráficos de 8 bits que compartan la misma paleta de colores. Fenix usará automáticamente la paleta de colores del primer gráfico en recuperar de disco e ignorará la paleta de los demás. Incluso en modo de 16 bits existe una paleta de colores (si bien en ese modo la paleta sólo afecta a gráficos de 16 bits).
 
VEA TAMBIÉN
LOAD_PNG Recupera de disco un gráfico PNG
LOAD_PCX Recupera de disco un gráfico PCX
UNLOAD_MAP Libera la memoria ocupada por un gráfico
