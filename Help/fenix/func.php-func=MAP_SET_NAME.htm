MAP_SET_NAME ( LIBRERÍA, GRÁFICO, "NOMBRE" )

Cambia el nombre de un gráfico

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico dentro de la librería
STRINGNOMBRE : Nuevo nombre del gráfico
DESCRIPCIÓN

Todo gráfico tiene un nombre de hasta 32 caracteres, que puede cambiarse mediante esta función, o recuperarse mediante MAP_NAME . Este nombre es especialmente útil cuando el gráfico forma parte de una librería FPG, ya que de otra manera no habría otro sistema aparte del número del gráfico para distinguir uno de otro.

VEA TAMBIÉN
MAP_NAME Devuelve el nombre de un gráfico
