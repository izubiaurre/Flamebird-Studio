INT CONTROL_INFO ( CONTROL, PARAMETRO )

Devuelve información acerca de un control

PARÁMETROS
INTCONTROL : Identificador de un control
INTPARAMETRO : Tipo de parámetro requerido
VALOR DE RETORNO

INT   :   Valor del parámetro requerido

DESCRIPCIÓN

Devuelve un valor entero informativo sobre el control indicado como primer parámetro. El tipo de información que se solicita viene dado por el segundo parámetro, que debe ser una de las siguientes constantes definidas en GUI.INC:

  • CONTROL_WIDTH: Ancho en pixels del control
  • CONTROL_HEIGHT: Alto en pixels del control
  • CONTROL_X: Coordenada horizontal dentro del marco de la ventana
  • CONTROL_Y: Coordenada vertical dentro del marco de la ventana
  • CONTROL_HIDDEN: 1 si el control está oculto
  • CONTROL_MIN_WIDTH: Ancho mínimo del control
  • CONTROL_MAX_WIDTH: Ancho máximo del control
  • CONTROL_MIN_HEIGHT: Alto mínimo del control
  • CONTROL_MAX_HEIGHT: Alto máximo del control
  • CONTROL_VRESIZABLE: 1 si el control puede redimensionarse verticalmente
  • CONTROL_HRESIZABLE: 1 si el control puede redimensionarse horizontalmente

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
