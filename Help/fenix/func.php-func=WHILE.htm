Sentencia de bucle general

PARÁMETROS
expresióncondición : Condición para seguir ejecutando el bucle
DESCRIPCIÓN

Esta sentencia permite crear un bucle, es decir, una secuencia de código que se ejecuta repetidamente hasta que una condición deja de ser cierta.

La condición se evalúa antes de ejecutar el bucle por primera vez y, si ya de entrada es falsa, las sentencias contenidas hasta el END no se ejecutarán ni una sóla vez. En caso contrario, las sentencias se ejecutarán repetidamente, evaluándose de nuevo la expresión después de cada ejecución para comprobar si es preciso volver a ejecutar el bloque.

NOTAS
En Fenix, cualquier valor entero impar se considera cierto (como por ejemplo, 1) mientras cualquier valor par se considera falso (incluyendo el 0). Los operadores de comparación como == o > devuelven 1 ó 0 siempre, pero no hay razón para no poder usar cualquier otro cálculo.
