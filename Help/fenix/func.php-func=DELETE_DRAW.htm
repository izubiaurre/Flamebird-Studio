DELETE_DRAW ( OBJETO )

Borra un objeto gráfico dibujado con una de las funciones de primitivas gráficas

PARÁMETROS
INTOBJETO : Identificador del objeto a eliminar
DESCRIPCIÓN

Destruye un objeto creado con una de las funciones de primitivas (utilizan todas el prefijo DRAW_) cuando se crearon después de una llamada a DRAWING_Z . El objeto no será dibujado en posteriores frames, y la memoria utilizada para almacenarlo es liberada.

El parámetro que identifica al objeto es el valor devuelto por la función de primitiva gráfica llamada para crearlo. Alternativamente, puede especificarse 0 como objeto, con lo que esta función destruirá todos los objetos creados con funciones de primitivas gráficas.

VEA TAMBIÉN
DRAWING_MAP Escoge el gráfico destino de las primitivas gráficas
DRAWING_Z Activa el modo persistente en las funciones de dibujo de primitivas y escoge el nivel de profundidad a utilizar
DRAW_BOX Dibuja un rectángulo con relleno
DRAW_CIRCLE Dibuja una círculo
DRAW_FCIRCLE Dibuja un círculo con relleno
DRAW_LINE Dibuja una línea
DRAW_CURVE Dibuja una curva
DRAW_RECT Dibuja un rectángulo
