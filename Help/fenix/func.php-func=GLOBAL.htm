Declara las variables de alcance global

DESCRIPCIÓN

Esta sección contiene declaraciones de variables que identifican los datos de alcance global. Estas variables están accesibles en todo momento y en cualquier proceso y su valor es único (si un proceso modifica una variable global, el próximo que la lea verá ya el nuevo valor de la misma).

Consulta la sección Variables para obtener más información acerca de la sintaxis a usar para declarar una variable, y Tipos de dato para saber qué clases de variable hay disponibles.

Sólo hay una copia de cada variable global, en todo momento, a diferencia de las variables privadas o locales, de las que hay varias versiones de la misma variable (una por cada proceso que las utilice).

Es posible que la sección GLOBAL aparezca varias veces en el mismo programa. Las variables globales declaradas en las secciones siguientes son igualmente válidas y se añaden a las declaradas anteriormente. Sin embargo, una variable debe declararse siempre antes de ser usada.

Variables globales predefinidas

Además de las variables que declares en tu programa, tienes disponibles una serie de variables del sistema predefinidas a las que puedes acceder en todo momento. Estas variables contienen información útil sobre el entorno de ejecución de tu programa y otros datos del sistema. Puedes acceder Variables globales aquí a una lista y descripción de estas variables.
