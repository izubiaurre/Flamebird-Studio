TABLE_ADDCOLUMN_BITMAPS ( TABLA, "NOMBRE" )

Añade una columna a una tabla, opcionalmente con datos

PARÁMETROS
INTTABLA : Identificador devuelto por TABLE
STRINGNOMBRE : Nombre de la columna
DESCRIPCIÓN

Añade una columna a una tabla, al final de las existentes (si las hay). No se incluyen datos en la columna, pero se marca la misma para contener gráficos en lugar de cadenas.

Una columna que contiene gráficos espera encontrar el número de cada gráfico como dato de cada fila cuando se añade ésta con la función TABLE_ADDLINE . La librería a la cual pertenezca el gráfico debe ser la misma para toda la columna: por defecto la 0, pero es posible cambiarla mediante la función TABLE_LIBRARY .

VEA TAMBIÉN
TABLE Crea una tabla o lista
TABLE_REMOVECOLUMN Quita una columna de una tabla
