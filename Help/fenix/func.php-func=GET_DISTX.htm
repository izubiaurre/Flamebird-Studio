GET_DISTX ( ÁNGULO, DISTANCIA )

Devuelve el ancho de un desplazamiento

PARÁMETROS
INTÁNGULO : Ángulo en milésimas de grado (90000 = 90º)
INTDISTANCIA : Magnitud de distancia
DESCRIPCIÓN

Esta función devuelve, en las misma unidad que uses para la distancia, el ancho del rectángulo formado por la línea que recorre esa distancia. Usando esta función junto con GET_DISTY puedes saber qué cantidades sumar a las coordenadas X e Y de un objeto para desplazarlo en la dirección que desees.
