WINDOW_CHANGED ( VENTANA, [ CAMBIADA ] )

Recupera o cambia el flag de "modificada" de una ventana

PARÁMETROS
INTVENTANA : Identificador de la ventana
INT CAMBIADA : (opcional)   Nuevo valor para el flag
VALOR DE RETORNO

INT   :   Valor del indicador de "modificada" de la ventana

DESCRIPCIÓN

Todas las ventanas disponen de un "flag" o indicador interno para especificar si su contenido ha sido modificado por el usuario y aún no se ha grabado a disco. Las ventanas con este flag activado muestran un asterisco (*) a la derecha de su título.

Es responsabilidad del programa asignar este indicador a las ventanas a medida que el usuario realice cualquier cambio en su estado, así como comprobar que una ventana no sea cerrada con información no guardada en disco, sin pedir antes confirmación al usuario.

Si no se especifica el segundo parámetro, esta función se usa para obtener el actual valor del indicador. Si desea cambiarse basta con indicar el nuevo valor (TRUE o FALSE) como segundo parámetro.

VEA TAMBIÉN
WINDOW Crea una nueva ventana
