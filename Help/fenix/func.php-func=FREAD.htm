FREAD ( FICHERO, VARIABLE )

Recupera datos de un fichero abierto

PARÁMETROS
INTFICHERO : Identificador de fichero devuelto por FOPEN
VARVARIABLE : Nombre de la variable a leer
VALOR DE RETORNO

INT   :   Número de bytes leídos, 0 en caso de error

DESCRIPCIÓN

Esta función lee datos binarios de un fichero abierto con la función FOPEN . Esta función sólo puede leer una variable a la vez. Sin embargo, esa variable puede ser un array o estructura: se leerán secuencialmente todos los elementos del array o estructura, por orden.

Los datos se leen en modo binario. Por ejemplo, leer una variable de tipo INT leerá cuatro bytes del disco. Al leer datos enteros de más de un byte, se supone un formato INTEL y se ordenará el dato en otras plataformas, por lo que los ficheros binarios escritos por FWRITE pueden leerse con más facilidad.

En el caso de escribir a disco una cadena STRING, se leerá en primer lugar un entero de cuatro bytes con el número de caracteres de la misma, y a continuación todos los caracteres en sí. Esto significa que el tamaño de un fichero que contenga cadenas es variable.

Si se desea procesar ficheros binarios genéricos o escritos con otra utilidad, puede usarse esta función para leer un dato de tipo BYTE cada vez, y procesarlos en secuencia.

VEA TAMBIÉN
FOPEN Abre un fichero en disco
FWRITE Guarda datos en un fichero abierto
FSEEK Mueve el puntero de lectura/escritura en un fichero
FILE_EXISTS Determina si el fichero existe o no en la ruta especificada
