INT INPUTLINE ( ANCHO, VARIABLE )

Crea una caja de Input de una sóla línea

PARÁMETROS
INTANCHO : Ancho en caracteres
VARVARIABLE : Variable de tipo CHAR[] con el texto
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea una caja de Input, donde el usuario puede escribir un texto, de un ancho determinado (el ancho especificado a la función vendrá dado en caracteres, no en pixels; el ancho real del control será aproxiamdamente esta cantidad más 1, multiplicada por 8). Como segundo parámetro se especificará una variable (directamente, sin usar el operador & ) que deberá ser de tipo CHAR[] (una cadena de ancho limitado). El control se asegurará de esta manera que el usuario no escriba más caracteres de los que caben en la memoria disponible para ello.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
