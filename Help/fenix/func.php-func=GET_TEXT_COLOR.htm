GET_TEXT_COLOR

Devuelve el color con que se escribirán los textos.

DESCRIPCIÓN

Esta función nos devolverá el valor de color con que se escribirán los textos dibujados con WRITE y derivados. En el caso de 8 bits devuelve un índice de la paleta, mientras que en 16 bits devuelve un entero para el RGB que puede descomponerse en sus componentes usando GET_RGB .

VEA TAMBIÉN
SET_TEXT_COLOR Escoge el color con el que se dibujarán los textos de funciones como
WRITE Dibuja un texto en pantalla
WRITE_INT 
WRITE_FLOAT 
WRITE_STRING 
WRITE_VAR Dibuja un texto en pantalla, con el contenido de una variable
WRITE_IN_MAP Crea un gráfico que contenga un texto
