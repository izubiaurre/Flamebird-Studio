Declara las variables de alcance local

DESCRIPCIÓN

Esta sección contiene declaraciones de variables que identifican los datos de alcance local. Todos los procesos contienen una copia propia de todas las variables locales. El valor de una variable local puede ser (y seguramente será) diferente de un proceso a otro. Las variables locales identifican propiedades de un proceso, como su aspecto o posición en pantalla. Además, las variables locales indican propiedades de utilidad general para todos los procesos y no sólo para procesos de uno u otro tipo.

Todos los procesos, independientemente de que las usen o no, reservarán espacio para todas las variables locales y las inicializarán. Es conveniente que limites al máximo el número de variables locales. Si una variable sólo es útil dentro de un proceso o unos pocos y no es necesario consultar su valor desde otros procesos, usa una variable privada en su lugar.

Consulta la sección Variables para obtener más información acerca de la sintaxis a usar para declarar una variable, y Tipos de dato para saber qué clases de variable hay disponibles.

Es posible que la sección LOCAL aparezca varias veces en el mismo programa. Las variables locales declaradas en las secciones siguientes son igualmente válidas y se añaden a las declaradas anteriormente. Sin embargo, una variable debe declararse siempre antes de ser usada.
