Declara una variable tipo estructura

DESCRIPCIÓN

Una variable tipo STRUCT es un dato compuesto por otros datos: en lugar de contener un dato simple, contiene múltiples datos de cualquier tipo (incluso otras estructuras). STRUCT viene seguido de un bloque acabado en END, que contiene una declaración de con la misma sintaxis que una sección de datos normal.

En el programa, es posible acceder al contenido de una estructura usando el operador "." (si la estructura se llama PERSONA y contiene datos como NOMBRE o APELLIDOS, se accedería a su contenido con la sintaxis PERSONA.NOMBRE, por ejemplo).

Es posible crear un array de estructuras en lugar de una sóla estructura, añadiendo dimensiones inmediatamente después del nombre. Por ejemplo, el siguiente código: 

 STRUCT colores[9] 

 BYTE r, g, b; 

 END
crea diez estructuras de nombre "colores", cada una de ellas con tres datos de tipo BYTE en el interior. Para acceder a los datos de cada uno de los miembros del array usaríamos la sintaxis habitual (colores[3].r por ejemplo).

Inicialización

Es posible otorgar valores a los datos en el bloque, usando la sintaxis normal en una sección de datos (= valor inmediatamente después del nombre de la variable. Sin embargo, en el caso de un array esto asignaría el mismo valor a la variable dentro de todos los elementos del array.

Alternativamente es posible inicializar la estructura escribiendo "=" justo después del END y seguirlo de los valores a asignar a cada uno de los miembros en el mismo orden en que fueron declarados, separando los valores por comas. Si es un array, se pueden incluir tantos valores como entradas en el array haya, proporcionando valores para todas las variables que contienen.
