DRAWING_COLOR ( COLOR )

Escoge el color con el que se dibujarán las primitivas gráficas con funciones como

PARÁMETROS
INTCOLOR : Número de color
DESCRIPCIÓN

Esta función permite escoger el color con el que se dibujarán las primitivas gráficas (rutinas que empiezan por DRAW...) así como los gráficos de dos colores creados por NEW_MAP .

En el caso de modos gráficos de 256 colores (8 bits), el valor de color debe ser un número de 0 a 255. En el caso de modos gráficos de 16 bits, el valor es una codificación de las componentes del color que varía en función de la tarjeta de vídeo y el modo gráfico. Lo normal es usar la función RGB para obtener la codificación de un color concreto, o bien usar un color obtenido por una función como MAP_GET_PIXEL .

Sin embargo, es posible usar DRAWING_MAP para dibujar en un gráfico de 8 bits incluso en modo de 16 bits. En estos casos, las primitivas gráficas usarán un color aproximado al especificado por esta función, como si se hubiese buscado con la función FIND_NEAREST_COLOR . En cambio, no es válido dibujar sobre un gráfico de 16 bits en un modo de 8 bits, porque la configuración de la tarjeta gráfica determina el formato exacto del color en 16 bits.

VEA TAMBIÉN
DRAWING_MAP Escoge el gráfico destino de las primitivas gráficas
DRAW_BOX Dibuja un rectángulo con relleno
DRAW_CIRCLE Dibuja una círculo
DRAW_FCIRCLE Dibuja un círculo con relleno
DRAW_LINE Dibuja una línea
DRAW_RECT Dibuja un rectángulo
