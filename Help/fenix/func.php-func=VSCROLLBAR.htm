INT VSCROLLBAR ( ALTO, &POSICIÓN, [ ESTILO ] )

Crea una barra de desplazamiento vertical

PARÁMETROS
INTALTO : Alto en pixels del control
POINTERPOSICIÓN : Puntero a la estructura con datos de posición de la barra
INT ESTILO : (opcional)   Estilo del control
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea una barra de desplazamiento vertical. La barra de desplazamiento puede tener el mismo aspecto que las que aparecen automáticamente al redimensionar algunas ventanas, pero es posible personalizarlo con diversas opciones.

El segundo parámetro es un puntero a una estructura de tipo ScrollBarInfo, definida en GUI.INC, y que cuenta con los siguientes campos:

  • INT Min: valor mínimo de posición.
  • INT Max: valor máximo de posición.
  • INT SingleStep: valor mínimo de incremento, efectivo cuando el usuario arrastra el control o hace clic en los botones laterales de la barra.
  • INT PageStep: valor de incremento ampliado, efectivo cuando el usuario hace clic en la zona de la barra encima o debajo del control.
  • INT Pos: posición actual de la barra (un valor cualquiera entre Min y Max.
Cuando el usuario actúe sobre el control, éste actualizará automáticamente el valor de Pos, pero nunca modificará el resto.

El tercer parámetro, opcional, permite personalizar el estilo de la barra de desplazamiento. Es una suma de cero o más de las siguientes constantes predefinidas en GUI.INC:

  • SCROLLBAR_BUTTONS: el control tiene dos botones, superior e inferior, con flechas.
  • SCROLLBAR_FRAME: el control tiene un recuadro dibujado a su alrededor, y un fondo.
  • SCROLLBAR_FOCUSABLE: el usuario puede hacer foco en la barra de desplazamiento y cambiar su valor con las teclas de cursor.
  • SCROLLBAR_FIXED: la barra de desplazamiento debe preservar siempre el valor de MinStep, siendo imposible marcar posiciones intermedias. Además, la barra de desplazamiento tendrá un aspecto alternativo, para marcar más claramente las diferencias con una barra normal.
El estilo por defecto, si no se especifica ninguno, es SCROLLBAR_BUTTONS + SCROLLBAR_FRAME.
