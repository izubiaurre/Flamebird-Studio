SET_SONG_VOLUME ( VOLUMEN )

Permite establecer el volumen general de la música

PARÁMETROS
INTVOLUMEN : Volumen deseado (0-128)
DESCRIPCIÓN

Esta función cambia el nivel total de volumen para la música entre un nivel de 0 (silencio total) y 128 (volumen original del fichero).

VEA TAMBIÉN
FADE_MUSIC_IN Reproduce música subiendo gradualmente el volumen desde cero
FADE_MUSIC_OFF Detiene la música bajando gradualmente el volumen
PLAY_SONG Inicia la reproducción de música
LOAD_SONG Recupera de disco un fichero de música
