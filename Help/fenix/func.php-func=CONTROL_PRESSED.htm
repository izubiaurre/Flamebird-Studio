INT CONTROL_PRESSED ( CONTROL, BOTÓN )

Comprueba si el usuario hizo clic recientemente sobre el control

PARÁMETROS
INTCONTROL : Identificador de un control cualquiera
INTBOTÓN : Número de botón
VALOR DE RETORNO

INT   :   1 si el usuario hizo click sobre el control recientemente

DESCRIPCIÓN

Cada vez que el usuario hace click sobre un control cualquiera, se activa un flag interno en el control. Esta función devuelve el flag correspondiente al botón deseado y lo pone a 0, de manera que en posteriores llamadas sólo devolverá 1 si el usuario ha vuelto a hacer click sobre el control.

Esta función se ha diseñado para ejecutarse cada frame, como medio para detectar si el usuario interactúa contra algún control.

VEA TAMBIÉN
CONTROL_DOUBLECLICK Comprueba si el usuario hizo doble clic recientemente sobre el control
CONTROL_CHANGED Comprueba si el usuario modificó recientemente el contenido del control
