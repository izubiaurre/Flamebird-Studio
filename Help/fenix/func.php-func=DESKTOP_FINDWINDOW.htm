INT DESKTOP_FINDWINDOW ( X, Y )

Busca una ventana en una posición de pantalla

PARÁMETROS
INTX : Coordenada horizontal
INTY : Coordenada vertical
VALOR DE RETORNO

INT   :   Identificador de la ventana encontrada, ó 0 si no hay ninguna

DESCRIPCIÓN

Esta función devuelve el identificador de la ventana que ocupe el punto de pantalla especificado por las coordenadas x e y. Si hay varias ventanas en ese punto, el identificador de la que se encuentre más alta (más cerca del usuario) será el devuelto. Si no hubiese ninguna ventana tocando dicha posición, esta función devolverá 0.
