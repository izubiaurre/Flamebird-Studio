STRING JOIN ( "SEPARADOR", &ARRAY, TAMAÑO )

Une una serie de cadenas

PARÁMETROS
STRINGSEPARADOR : Cadena a intercalar entre elementos
STRING POINTERARRAY : Array con los elementos a unir
INTTAMAÑO : Número de elementos en el array
VALOR DE RETORNO

STRING   :   Cadena resultante

DESCRIPCIÓN

Dado un array de cadenas STRING, esta función junta todas ellas en una cadena final, intercalando la cadena especificada como separador entre los elementos. Esta función permite obtener el efecto contrario a la función SPLIT .

Por ejemplo, si un array contiene tres cadenas: "12", "04" y "1980", y se utiliza la cadena "/" como separador, esta función devolverá la cadena "12/04/1980".

VEA TAMBIÉN
SPLIT Divide una cadena en partes
