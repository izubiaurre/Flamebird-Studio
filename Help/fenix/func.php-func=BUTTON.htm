INT BUTTON ( "TEXTO", [ PROCESO ] )

Crea un botón

PARÁMETROS
STRINGTEXTO : Texto del botón
INT PROCESO : (opcional)   Tipo del proceso a crear
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea un botón, un control que contiene un texto y el usuario puede pulsar para ejecutar una acción determinada. Es posible especificar, como segundo parámetro, un tipo de proceso empleando el operador TYPE. El botón creará un proceso de este tipo cada vez que sea pulsado y lo ejecutará inmediatamente. Este proceso no deberá recibir parámetros.

Si no se desea crear una multitud de procesos, uno por cada botón creado, una alternativa es usar la función BUTTON_PRESSED en un bucle con FRAME, para comprobar si el usuario lo pulsa.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
BUTTON_PRESSED Comprueba si el botón ha sido pulsado
