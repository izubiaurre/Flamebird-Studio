DRAWING_MAP ( LIBRERÍA, GRÁFICO )

Escoge el gráfico destino de las primitivas gráficas

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico a dibujar
DESCRIPCIÓN

Esta función permite escoger el gráfico donde se dibujarán las primitivas gráficas (todas aquellas funciones que empiezan por DRAW_...). No es válido llamar a ninguna de estas funciones si no se escoge antes un fondo como destino. El gráfico (0, 0) representa el fondo de pantalla.

VEA TAMBIÉN
DRAWING_COLOR Escoge el color con el que se dibujarán las primitivas gráficas con funciones como
DRAW_BOX Dibuja un rectángulo con relleno
DRAW_CIRCLE Dibuja una círculo
DRAW_FCIRCLE Dibuja un círculo con relleno
DRAW_LINE Dibuja una línea
DRAW_RECT Dibuja un rectángulo
