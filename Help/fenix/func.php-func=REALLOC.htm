POINTER REALLOC ( &MEM, BYTES )

Amplía o reduce el espacio reservado de un bloque de memoria

PARÁMETROS
POINTERMEM : Puntero a una zona de memoria reservada en una llamada anterior de
INTBYTES : Tamaño en bytes deseado para la zona de memoria
DESCRIPCIÓN

REALLOC amplía o reduce el espacio reservado de un bloque de memoria. Para ello recibe dos parámetros: el primero debe ser el puntero devuelto por ALLOC al reservar el bloque, el segundo será el nuevo tamaño que debe tener el bloque de memoria.

REALLOC puede cambiar la ubicación del bloque de memoria si fuese necesario. REALLOC devolverá un puntero al comienzo del nuevo bloque de memoria, que debe ser utilizado en lugar del que anteriormente había devuelto ALLOC, que deja de ser válido.

Los datos del bloque original que quepan en el nuevo bloque se mantendrán íntegramente, los que no quepan se descartarán, siempre desde el final del bloque, y si el nuevo tamaño es mayor la zona de memoria adicional puede contener valores aleatorios o procedentes de usos anteriores, siendo responsabilidad del programador inicializarla con los valores adecuados.
