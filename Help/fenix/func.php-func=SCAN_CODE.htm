SCAN_CODE

Contiene el código de tecla pulsado en el último frame

DESCRIPCIÓN

Este valor se corresponde con el de las constantes predefinidas para la función KEY . Al igual que ASCII si se ha pulsado más de una tecla las pulsaciones adicionales quedan almacenadas en un pequeño buffer para evitar su pérdida.

Es recomendable utilizar KEY para hacer la comprobación de qué tecla se ha pulsado, ya que es más rápido que hacer la comprobación manual a traves de código en nuestro programa.
