INCLUDE " fichero "

Inserta un fichero de código fuente en mitad de otro

PARÁMETROS
STRINGfichero : Nombre de fichero a incluir
DESCRIPCIÓN

Esta directiva hace que el compilador lea el contenido de otro fichero (cuyo nombre se indica entre comillas justo después de la palabra INCLUDE) en la posición donde se encuentra la directiva. El fichero incluído puede tener cualquier extensión, pero se recomienda .INC .

Puede usarse en cualquier lugar del fichero, pero se recomienda utilizarla fuera de cualquier sección como norma general.

Para dividir un programa extenso en varios ficheros, la forma recomendada de trabajar es crear varios ficheros .INC, cada uno de ellos con varios procesos relacionados, y secciones GLOBAL o LOCAL con las variables que utilicen. El fichero principal del juego constará de una serie de INCLUDE con todos estos ficheros secundarios, y el proceso principal. Dado que las secciones de datos pueden aparecer varias veces en un programa, es posible dividirlas entre varios ficheros.
