FILE_EXISTS ( "FILE" )

Determina si el fichero existe o no en la ruta especificada

PARÁMETROS
STRINGFILE : Nombre del fichero
DESCRIPCIÓN

Esta función comprueba si un fichero existe, devuelve 1 si el fichero existe y 0 si no existe. El fichero se busca en la ruta actual por defecto o en la ruta que se haya especificado como parámetro.
