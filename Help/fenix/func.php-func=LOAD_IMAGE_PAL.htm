LOAD_IMAGE_PAL ( "FICHERO" )

Recupera la paleta de una imagen de disco detectando automáticamente su tipo mediante SDL_image.DLL

PARÁMETROS
STRINGFICHERO : Nombre de un fichero de imagen
DESCRIPCIÓN

Esta función recupera de disco la paleta contenida por una imagen de 8 bits en cualquiera de los formatos soportados por LOAD_IMAGE.

La función LOAD_IMAGE cargará automáticamente la paleta del primer gráfico de 8 bits que se recupere en el programa. Sin embargo, para programas que trabajan con más de una paleta de colores, es necesario utilizar LOAD_IMAGE_PAL para recuperar la paleta de colores o las imagenes de 8 bits no se mostrarán correctamente.

La función soporta cualquier formato reconocido por LOAD_IMAGE (siempre que las imágenes sean de 8 bits y con paleta de colores).

NOTAS

Si la imagen utiliza una paleta de menos de 256 colores, sólo los primeros colores (0 - n) de la paleta actual serán modificados.

 
VEA TAMBIÉN
LOAD_PAL Recupera de disco una paleta de colores
LOAD_IMAGE Recupera una imagen de disco detectando automáticamente su tipo, utilizando SDL_image.DLL
