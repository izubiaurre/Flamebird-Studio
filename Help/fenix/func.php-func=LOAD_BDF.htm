LOAD_BDF ( "FICHERO" )

Recupera de disco un tipo de letra BDF

PARÁMETROS
STRINGFICHERO : Nombre del fichero
VALOR DE RETORNO

INT   :   Identificador con el nuevo tipo de letra

DESCRIPCIÓN

Esta función recupera un tipo de letra BDF desde disco. El BDF es un formato de tipo de letra creado por Adobe en uso en sistemas operativos Unix. Es un tipo de letra a dos colores, por lo que es posible usar la función SET_TEXT_COLOR para dibujar textos en varios colores diferentes.

LOAD_BDF devuelve un número entero que identifica al nuevo tipo de letra. Es posible utilizar a partir de entonces este valor con cualquier función que admita un tipo de letra como parámetro, como por ejemplo WRITE .

VEA TAMBIÉN
WRITE Dibuja un texto en pantalla
WRITE_IN_MAP Crea un gráfico que contenga un texto
UNLOAD_FNT Libera la memoria ocupada por un tipo de letra
