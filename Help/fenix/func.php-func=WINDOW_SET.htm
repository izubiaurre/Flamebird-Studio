WINDOW_SET ( VENTANA, TIPO, VALOR )

Permite cambiar ciertos parámetros internos de una ventana

PARÁMETROS
INTVENTANA : Identificador de la ventana
INTTIPO : Tipo de parámetro
INTVALOR : Nuevo valor para el parámetro
DESCRIPCIÓN

Esta función permite alterar el aspecto o funcionalidad de una ventana, en tiempo de ejecución. El segundo parámetro debe ser una de las siguientes constantes predefinidas en GUI.INC:

  • WINDOW_ONTOP: Con un valor de TRUE en este parámetro, la ventana permanecerá siempre por encima del resto, incluso de la que tenga el foco del ratón.
  • WINDOW_MOVEABLE: Por defecto este parámetro es TRUE, pero con un valor de FALSE el usuario no podrá desplazar la ventana, que quedará bloqueada en el sitio.
  • WINDOW_STYLE: Asignar un valor a este parámetro permite modificar el estilo de la ventana en tiempo de ejecución. Mira la función WINDOW para obtener una lista de los estilos de ventana disponibles.

VEA TAMBIÉN
WINDOW Crea una nueva ventana
