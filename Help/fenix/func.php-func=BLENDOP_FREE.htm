BLENDOP_FREE ( BLENDOP )

Destruye una tabla blendop

PARÁMETROS
INTBLENDOP : Identificador de una tabla BLENDOP
DESCRIPCIÓN

Esta función libera la memoria ocupada por una tabla blendop. Es responsabilidad del programador asegurarse de que la tabla blendop no se encuentre asignada a ningún gráfico ni a ninguna variable BLENDOP de ningún proceso en el momento de destruírla. Lo contrario se considera una condición de error grave y puede dar resultados impredecibles.

VEA TAMBIÉN
BLENDOP_NEW Crea una nueva tabla BLENDOP
BLENDOP_IDENTITY Reinicia una tabla blendop
BLENDOP_TRANSLUCENCY Aplica transparencia a una tabla blendop
BLENDOP_TINT Aplica un color a una tabla blendop
BLENDOP_INTENSITY Aplica transparencia a una tabla blendop
BLENDOP_GRAYSCALE Aplica conversión a escala de grises a una tabla blendop
BLENDOP_SWAP Intercambia el efecto de una operación en una tabla blendop
BLENDOP_ASSIGN Asigna una tabla blendop a un gráfico
BLENDOP_APPLY Aplica una tabla blendop a los pixels de un gráfico
