Declara las variables privadas de un proceso

DESCRIPCIÓN

Esta sección contiene declaraciones de variables privadas de un proceso concreto. Estas variables son propias del proceso: ningún otro proceso tiene acceso a ellas y sólo procesos del mismo tipo exactamente al actual guardan una copia de las mismas. El valor de una variable privada puede ser (y seguramente será) diferente de un proceso que la tenga a otro. Las variables privadas identifican datos temporales o de propósito concreto dentro de un proceso, como un contador o un valor de estado que no tiene relevancia para el resto de procesos.

Consulta la sección Variables para obtener más información acerca de la sintaxis a usar para declarar una variable, y Tipos de dato para saber qué clases de variable hay disponibles.

La sección PRIVATE aparece dentro de cada proceso y es opcional. Un proceso que no la use no dispone de variables privadas, aunque sí tiene variables locales (acude a la sección LOCAL para más información).

A diferencia de las variables locales o globales, no existen variables privadas predefinidas. Un proceso por defecto no tiene ninguna variable privada hasta que no se declare alguna.
