INT RADIO ( "TEXTO", &VARIABLE, VALOR )

Crea un botón de "radio"

PARÁMETROS
STRINGTEXTO : Texto de la caja de chequeo
POINTERVARIABLE : Puntero a la variable INT
INTVALOR : Contenido a comparar
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea un botón de radio, un control con dos estados: activado y desactivado, que el usuario puede cambiar y que se basan en el contenido de una variable INT, cuya dirección se le pasará a esta función como segundo parámetro empleando el operador & . Si esta variable contiene exactamente el valor que se pasa a esta función como tercer parámetro, el botón estará marcado. El control cambiará el contenido de la variable automáticamente cuando el usuario manipule el control.

Este control se diferencia del control CHECKBOX en que está diseñado para crear varios botones que se refieran a la misma variable INT con distintos valores, de forma que el usuario puede hacer una elección entre más de dos opciones.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
