GET_POINT ( LIBRERÍA, GRÁFICO, NÚMERO, &X, &Y )

Permite obtener un punto de control de un gráfico

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico dentro de la librería
INTNÚMERO : Número de punto de control (0 es el centro)
INT POINTERX : Offset de una variable de tipo entero (INT) que pasará a contener la coordenada horizontal del punto
INT POINTERY : Offset de una variable de tipo entero (INT) que pasará a contener la coordenada vertical del punto
VALOR DE RETORNO

INT   :   1 si el punto está definido, 0 en caso contrario

DESCRIPCIÓN

Un gráfico cualquiera puede contener un número indeterminado de puntos de control de los cuales el punto de control 0 representa el centro del gráfico, y el resto están disponibles para el usuario. Mediante esta función puede consultarse un punto de control del gráfico. Si el número de punto de control especificado queda fuera de rango o no está definido, la función devuelve 0 y las variables x e y no serán modificadas. En el resto de casos, x e y pasarán a contener las coordenadas del punto de control, de manera que (0, 0) representa el pixel ubicado en la esquina superior izquierda del gráfico, y las coordenadas crecen hacia abajo y a la derecha.

VEA TAMBIÉN
SET_POINT Permite cambiar un punto de control de un gráfico
GET_REAL_POINT Permite obtener la posición en pantalla de un punto de control
