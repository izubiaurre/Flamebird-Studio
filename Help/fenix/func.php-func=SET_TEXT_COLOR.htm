SET_TEXT_COLOR ( COLOR )

Escoge el color con el que se dibujarán los textos de funciones como

PARÁMETROS
INTCOLOR : Número de color
DESCRIPCIÓN

Esta función permite escoger el color de texto con el que se dibujarán los caracteres de los tipos de letra a dos colores. Los tipos de letra (fuentes) de 256 colores o de 16 bits ya incluyen información de color y no se verán afectados por esta función. Los tipos de letra a dos colores pueden recuperarse mediante las funciones LOAD_BDF o LOAD_FNT .

Los textos dibujados con WRITE y derivados recuerdan el color escogido en el momento que se llamaron, aunque dibujen su texto cada frame. Por lo tanto, es válido llamar a TEXT_COLOR antes de dibujar cada uno y poder mostrar en pantalla textos de varios colores simultáneamente.

En el caso de gráficos de 256 colores (8 bits), el valor de color debe ser un número de 0 a 255. En el caso de gráficos de 16 bits, el valor es una codificación de las componentes del color que varía en función de la tarjeta de vídeo y el modo gráfico. Lo normal es usar la función RGB para obtener la codificación de un color concreto, o bien usar un color obtenido por una función como MAP_GET_PIXEL .
