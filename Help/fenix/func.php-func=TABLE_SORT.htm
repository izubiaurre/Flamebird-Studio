TABLE_SORT ( TABLA, "NOMBRE", ASCENDENTE )

Ordena una tabla

PARÁMETROS
INTTABLA : Identificador devuelto por TABLE
STRINGNOMBRE : Nombre de la columna
INTASCENDENTE : Orden de la ordenación
DESCRIPCIÓN

Ordena una tabla a partir de la columna especificada como segundo parámetro. Se recomienda usar el nombre de la columna, aunque alternativamente es posible usar un número de índice, con 0 para la columna más a la izquierda. El tercer parámetro indica el orden de la operación, 1 para ascendente y 0 para descendente.

VEA TAMBIÉN
TABLE Crea una tabla o lista
