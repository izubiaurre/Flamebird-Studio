INT MENU

Crea un nuevo menú

VALOR DE RETORNO

INT   :   Identificador del menú creado

DESCRIPCIÓN

Crea un menú de opciones. Un menú de opciones no es un control, sino simplemente una lista de opciones en memoria. El menú puede usarse para crear distintos tipos de controles que en efecto muestran el menú en pantalla, como por ejemplo una barra de menús ( TOOLBAR ) o un menú pop-up ( POPUP_MENU ).

El menú creado por esta función está inicialmente vacío. Es preciso llenarlo de opciones mediante las funciones MENU_ACTION , MENU_SEPARATOR , MENU_TOGGLE , MENU_RADIO o MENU_SUBMENU .

El identificador devuelto por esta función debe utilizarse a la hora de crear el control que lo contiene, así como en cada una de las funciones anteriormente descritas.

VEA TAMBIÉN
MENU_ACTION 
MENU_TOGGLE Añade una línea con marca de chequeo a un menú
MENU_SEPARATOR Añade un separador a un menú Añade un separador al último menú creado
MENU_RADIO Añade una opción (con marca de chequeo) a un menú
