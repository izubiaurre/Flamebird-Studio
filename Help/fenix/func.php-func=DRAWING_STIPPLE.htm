DRAWING_STIPPLE ( VALOR )

Escoge el aspecto de dibujo de líneas

PARÁMETROS
INTVALOR : Nuevo valor de "stipple"
DESCRIPCIÓN

Esta función permite escoger el aspecto con el que se dibujan las líneas dibujadas con DRAW_LINE o DRAW_RECT . El parámetro es un número entero de 32 bits, en el cual cada bit a 1 representa un pixel activo en cada segmento de 32 pixels de la línea. Por defecto, el valor de stipple es 0FFFFFFFFh, con los 32 bits a 1, por lo que las líneas tienen la apariencia de una línea sólida.

Algunos valores útiles para este parámetro son 055555555h para dibujar una línea de puntos o 0F0F0F0Fh para dibujar una línea de rayas.

NOTAS
El primer punto a dibujar estará activo si lo está el bit más a la derecha del número. Este parámetro va rotando a la derecha a medida que se dibujan puntos, por lo que si varias líneas se dibujan una a continuación de la anterior, darán una impresión de continuidad de la apariencia de la línea.
 
VEA TAMBIÉN
DRAWING_MAP Escoge el gráfico destino de las primitivas gráficas
DRAW_LINE Dibuja una línea
DRAW_RECT Dibuja un rectángulo
