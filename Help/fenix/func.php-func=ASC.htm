ASC ( "CADENA" )

Devuelve el código ASCII de un carácter

PARÁMETROS
STRINGCADENA : Cadena con el carácter
VALOR DE RETORNO

INT   :   Código de caracter

DESCRIPCIÓN

Esta función devuelve el código ASCII (un número entre 0 y 255) que corresponde al primer carácter de la cadena especificada como parámetro. Por ejemplo, ASC("A") devuelve 64.

VEA TAMBIÉN
CHR Devuelve el carácter que representa el código ASCII dado
