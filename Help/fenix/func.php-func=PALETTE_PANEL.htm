INT PALETTE_PANEL ( ANCHO, ALTO, [ &VARIABLE ] )

Crea un control de paleta

PARÁMETROS
INTANCHO : Ancho en pixels del control
INTALTO : Alto en pixels del control
POINTER VARIABLE : (opcional)   Variable INT con el número de color seleccionado
VALOR DE RETORNO

INT   :   Identificador del control creado

DESCRIPCIÓN

Crea un control de paleta, que muestra los 256 colores de la paleta de colores actual.

Por defecto este es un control estático con el que no es posible interactuar, como una etiqueta de texto LABEL o una imagen IMAGE . Sin embargo, si se especifica el tercer parámetro con la dirección de una variable de tipo INT pasada mediante el operador & , entonces el control permite elegir un color concreto dentro de los 256, y modifica el contenido de la variable según el usuario haga clic en otro color.

VEA TAMBIÉN
WINDOW_ADDCONTROL Añade un control a una ventana
