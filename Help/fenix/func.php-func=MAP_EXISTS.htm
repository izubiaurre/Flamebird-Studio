MAP_EXISTS ( LIBRERÍA, GRÁFICO )

Devuelve 1 si un gráfico existe

PARÁMETROS
INTLIBRERÍA : Número de librería FPG
INTGRÁFICO : Número de gráfico dentro de la librería
DESCRIPCIÓN

Esta función devuelve 1 si un gráfico dado existe dentro de la librería indicada. Dado que sólo puede haber 1000 gráficos en una librería, con códigos de 0 a 999, es posible usar esta función dentro de un bucle para ennumerar todos los gráficos que existan dentro de una librería FPG.

VEA TAMBIÉN
LOAD_FPG Recupera de disco una librería FPG
