GUI_DRAGDESTINATION

Comprueba el destino de una operación de drag & drop

VALOR DE RETORNO

INT   :   Identificador del control de destino

DESCRIPCIÓN

Esta función permite comprobar qué control recibirá el destino de una operación de drag & drop. Sólo es válido llamarla cuando se ha comprobado que existe alguna en curso mediante la función GUI_DRAGGING . El valor devuelto será el identificador del control que haya bajo el ratón en este momento, o bien la constante especial DRAG_ATDESKTOP (definida en GUI.INC) para indicar que el usuario está arrastrando un objeto al escritorio. Si bajo el cursor no hay ningún control, esta función devolverá 0.

VEA TAMBIÉN
GUI_DRAGTYPE Devuelve qué tipo de objeto se está arrastrando
GUI_DRAGGING Comprueba si existe alguna operación de drag & drop
GUI_ACCEPTDRAG Activa la aceptación de una operación drag & drop
