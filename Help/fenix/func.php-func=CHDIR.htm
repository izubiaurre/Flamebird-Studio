CHDIR ( "DIRECTORIO" )

Cambia el directorio actual por el nuevo directorio

PARÁMETROS
STRINGDIRECTORIO : Nuevo directorio
DESCRIPCIÓN

Esta función cambia el directorio por defecto al que se especifica con el parámetro directorio . Devuelve 0 si ha tenido éxito y -1 en caso de no poderse efectuar el cambio al nuevo directorio, por ejemplo, si el directorio indicado no existe.

Las rutas pueden indicarse usando el separador de directorios / o \ y se puede especificar, en el caso de plataformas que así lo requieran, el nombre de unidad, por ejemplo c:\midirectorio en plataformas Windows.

VEA TAMBIÉN
CD Recupera el nombre del directorio actual
