#define CONSTANTE VALOR

Asigna un texto a una macro

PARÁMETROS
IDENTIFIERCONSTANTE : Nombre de constante
TEXTVALOR : Valor asignado a la constante
DESCRIPCIÓN

Esta directiva del preprocesador permite definir macros , palabras que al aparecer en el código serán sustituídas por el texto especificado en la directiva #define, antes de ser interpretadas por el compilador.

Estas macros no pueden recibir parámetros, como en C.

Es importante entender las diferencias entre las constantes definidas con CONST y las macros #DEFINE. Las constantes tienen un tipo determinado, y el compilador conoce su valor fijo final. Si se declara una constante con el valor "2+2", el compilador usará el valor 4. En cambio, una macro definida como "2+2" provocará la aparición del texto "2+2" en lugar de su nombre en el código.

Observa este ejemplo: 

 #define CALCULO 8+10 

 SAY (4*CALCULO);
El compilador en realidad lo entiende como: 
 SAY (4*8+10);
...lo caul seguramente no provoca el efecto deseado. Para evitar este tipo de problemas, usa paréntesis siempre que una macro pueda contener expresiones.

Otro uso importante de las macros consiste en emplearlas conjuntamente con #ifdef o #ifndef para provocar que determinadas secciones de código se compilen de forma opcional.

VEA TAMBIÉN
#IFDEF Provoca que una sección de código se compile sólo si una macro está definida
#IFNDEF Provoca que una sección de código se compile sólo si una macro no está definida
#IF Provoca que una sección de código se compile condicionalmente
CONST Declara las constantes del programa
