CONTROL_DRAGGABLE ( CONTROL, "TEXTO", [ FPG ], [ GRAPH ] )

Activa la opción drag & drop en un objeto

PARÁMETROS
INTCONTROL : Identificador de un control
STRINGTEXTO : Texto asociado
INT FPG : (opcional)   Librería FPG
INT GRAPH : (opcional)   Identificador de gráfico
DESCRIPCIÓN

Activa la posibilidad de arrastrar el contenido de un control. En algunos controles, esto afecta a toda su área. En otros, el usuario no podrá arrastrar algunas zonas internas (como barras de desplazamiento), pero sí el contenido principal del control.

Es preciso especificar una cadena de texto que sirva para identificar el objeto que se arrastra. La información que pueda contener esta cadena es totalmente libre, y será devuelta por GUI_DRAGTYPE cuando el usuario esté arrastrando el objeto.

Es posible opcionalmente indicar un gráfico que será visualizado de forma transparente cuando el usuario arrastre el contenido del control. Por defecto, se usa un simple recuadro blanco y negro.

VEA TAMBIÉN
GUI_DRAGGING Comprueba si existe alguna operación de drag & drop
GUI_DRAGTYPE Devuelve qué tipo de objeto se está arrastrando
