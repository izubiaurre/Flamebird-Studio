STRING WINDOW_DATA ( VENTANA, [ "TYPE" ] )

Permite obtener o cambiar datos extra asociados a una ventana

PARÁMETROS
INTVENTANA : Identificador de la ventana
STRING TYPE : (opcional)   Nuevo dato extra de ventana
VALOR DE RETORNO

STRING   :   Nuevo (o antiguo) dato extra de ventana

DESCRIPCIÓN

Esta función permite asignar una cadena de "datos adicionales" a una ventana. Al igual que el tipo de ventana, el objetivo de esta cadena es ofrecer al usuario la facilidad de almacenar información útil por cada ventana.

Si no se especifica el segundo parámetro, la función simplemente devuelve los datos actuales asociados a la ventana, sin cambiar nada. Si en cambio se especificase un dato, éste sustituiría al valor actual asociado a la ventana.

VEA TAMBIÉN
WINDOW_TYPE Permite obtener o cambiar el tipo de una ventana
