El preprocesador

El preprocesador es un pequeño programa que filtra el contenido de tu programa antes de pasarselo al compilador. Interpreta ciertas directivas que pueden estar presente en cualquier punto del programa y se caracterizan por ocupar una línea completa y empezar por el carácter # al principio de la misma (opcionalmente, precedido por espacios).

Las directivas del preprocesador permiten definir macros (palabras que, al aparecer más adelante de su definición en el código, son sustituídas por un texto determinado) o para crear secciones de código opcionales (que sólo aparecen si se cumple una condición determinada).

CONTENIDO
#DEFINE Asigna un texto a una macro
#IF Provoca que una sección de código se compile condicionalmente
#IFDEF Provoca que una sección de código se compile sólo si una macro está definida
#IFNDEF Provoca que una sección de código se compile sólo si una macro no está definida
