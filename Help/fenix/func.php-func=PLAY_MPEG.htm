PLAY_MPEG ( VÍDEO )

Inicia la reproducción de un vídeo

PARÁMETROS
INTVÍDEO : Identificador del vídeo
DESCRIPCIÓN

Esta función empieza a reproducir un vídeo MPEG. El vídeo no se muestra en pantalla automáticamente. Lo que hace esta función, en cambio, es devolver el código de un gráfico perteneciente a la librería 0 que representa en todo momento el frame actual del video.

Para mostrar el video en pantalla, la forma recomendada es asignar el código devuelto por PLAY_MPEG a la variable GRAPH de un proceso. Las coordenadas X e Y del proceso identificarán entonces la posición del video en pantalla, a partir de su centro.

Puedes usar el gráfico normalmente en todo tipo de operaciones. Por ejemplo, puedes usar la variable FLAGS para mostrar un video invertido, o transparente, o cambiarlo de tamaño con SIZE . Puedes usar también este gráfico en operaciones de dibujo como MAP_PUT o MAP_XPUT , pero ten en cuenta que en ese caso sólo accedes al último frame. El contenido del gráfico se reescribe contínuamente, por lo que no es posible modificarlo directamente (por ejemplo, para añadir un subtítulo). Si quieres llevar a cabo este tipo de operaciones, tendrás que dibujar los elementos adicionales en pantalla, con una coordenada Z superior, o bien copiar cada frame a otro gráfico antes de operar.

NOTAS
No se permite la reproducción de video MPEG en modo de 8 bits (256 colores).
 
VEA TAMBIÉN
