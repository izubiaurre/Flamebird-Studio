WRITE_VAR ( FUENTE, X, Y, ALINEACIÓN, VARIABLE )

Dibuja un texto en pantalla, con el contenido de una variable

PARÁMETROS
INTFUENTE : Tipo de letra
INTX : Coordenada horizontal
INTY : Coordenada vertical
INTALINEACIÓN : Tipo de alineación
VARVARIABLE : Variable a escribir
VALOR DE RETORNO

INT   :   Identificador del nuevo texto

DESCRIPCIÓN

Dibuja un texto en pantalla, en la posición indicada y haciendo uso del tipo de letra especificado (que puede ser 0 para indicar el tipo de letra del sistema, o bien un identificador devuelto por una función como LOAD_FNT ). El texto contendrá el valor que haya en todo momento en el interior de la variable local, privada o global especificada como último parámetro, y será redibujado cada frame, por lo que siempre contendrá una descripción exacta del valor. Es importante que la variable siga existiendo mientras lo haga el texto (por ejemplo, no es correcto usar una variable privada o local de un proceso que pueda ser destruído mientras el texto aún siga dibujándose).

El parámetro de alineación indica el significado de las coordenadas, ya que en función de su valor, el texto será alineado respecto a este punto como sigue:

  • 0: el punto ocupará la esquina superior izquierda del texto, una vez escrito
  • 1: el texto estará centrado horizontalmente sobre el punto, que ocupará la fila superior del mismo
  • 2: el punto ocupará la esquina superior derecha
  • 3: el texto estará centrado verticalmente sobre el punto, que ocupará la columna izquierda del mismo
  • 4: el texto estará centrado horizontal y verticalmente alrededor del punto
  • 5: el texto estará centrado verticalmente del punto, que ocupará la fila derecha del mismo
  • 6: el punto ocupará la esquina inferior izquierda
  • 7: el texto será centrado horizontalmente. El punto ocupará la fila inferior del mismo.
  • 8: el punto ocupará la esquina inferior derecha

Este texto será dibujado a cada frame, en el momento de profundidad indicado por la variable global TEXT_Z . Es importante entender que una nueva llamada a esta función incluso con los mismos parámetros crearía un segundo texto adicional a dibujar. Es preciso llamar a la función DELETE_TEXT cuando se desea que un texto ya no sea visible por más tiempo, teniendo cuidado con usar esta función dentro de un bucle.

Los tipos de letra de 1 bit por pixel pueden ser dibujados a cualquier color, escogiéndolo con la función SET_TEXT_COLOR . Los textos escritos con esta función recuerdan el color escogido en el momento que se llamaron, así que es válido dibujar varios textos a diferente color. El tipo de letra del sistema es un tipo de letra de 1 bit por pixel.

Esta función tiene una utilidad limitada, ya que es preciso utilizar una variable para contener el texto. Si se desea mostrar en pantalla un texto fijo y estático, es más práctico usar en su lugar la función WRITE .
