DESKTOP_REMOVEPOPUP ( CONTROL )

Quita de pantalla un control "popup" y lo destruye

PARÁMETROS
INTCONTROL : Identificador del control a eliminar
DESCRIPCIÓN

Esta función elimina un control "popup" que haya sido mostrado previamente mediante la función DESKTOP_ADDPOPUP . El control será destruído, y sus recursos de memoria liberados.

Es seguro llamar a esta función con un control que tal vez ya haya sido destruído automáticamente (por ejemplo, si el usuario hizo clic sobre otra ventana). En este caso, la función no tendrá ningún efecto.

Si se llama a esta función con el número 0 como identificador de control, todos los controles popup que haya presentes serán eliminados.

Normalmente esta función se llama cuando el usuario ya ha realizado cualquier tipo de elección con el control popup y éste no es útil por más tiempo. Algunos controles, como las listas o los menús, se eliminan automáticamente cuando el usuario escoge una opción y no es necesario llamar a esta función.

VEA TAMBIÉN
DESKTOP_ADDPOPUP Muestra un control "popup"
