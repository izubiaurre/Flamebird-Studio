INT DESKTOP_NEXTWINDOW ( VENTANA )

Dada una ventana, devuelve la siguiente

PARÁMETROS
INTVENTANA : Identificador de una ventana
VALOR DE RETORNO

INT   :   Identificador de la siguiente ventana, ó 0 si era la última

DESCRIPCIÓN

Devuelve la siguiente ventana, en la lista de ventanas presentes en el escritorio. Usando la función DESKTOP_BOTTOMWINDOW junto con esta, es posible recorrer todas las ventanas presentes en el escritorio. Esta función devuelve 0 si no hay más ventanas después de la presente.

Las ventanas se devuelven por orden de visibilidad. La última ventana devuelta es la que actualmente tiene el foco de entrada, la misma que devuelve la función DESKTOP_TOPWINDOW .
