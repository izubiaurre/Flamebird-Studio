GET_REAL_POINT ( NÚMERO, &X, &Y )

Permite obtener la posición en pantalla de un punto de control

PARÁMETROS
INTNÚMERO : Número de punto de control (0 es el centro)
INT POINTERX : Offset de una variable de tipo entero (INT) que pasará a contener la coordenada horizontal del punto
INT POINTERY : Offset de una variable de tipo entero (INT) que pasará a contener la coordenada vertical del punto
VALOR DE RETORNO

INT   :   1 si el punto está definido, 0 en caso contrario

DESCRIPCIÓN

Esta función permite obtener la posición en pantalla de un punto de control del gráfico contenido por el proceso actual, teniendo en cuenta las variables que indican el aspecto o posición del gráfico, como X , Y o ANGLE .

Las variables cuyo offset recibe esta función pasarán a contener las coordenadas del punto de control sobre la pantalla.

VEA TAMBIÉN
GET_POINT Permite obtener un punto de control de un gráfico
SET_POINT Permite cambiar un punto de control de un gráfico
