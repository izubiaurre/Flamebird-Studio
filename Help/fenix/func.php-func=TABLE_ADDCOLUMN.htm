TABLE_ADDCOLUMN ( TABLA, "NOMBRE", [ DATOS ] )

Añade una columna a una tabla, opcionalmente con datos

PARÁMETROS
INTTABLA : Identificador devuelto por TABLE
STRINGNOMBRE : Nombre de la columna
VAR DATOS : (opcional)   Array de STRING con datos
DESCRIPCIÓN

Añade una columna a una tabla, al final de las existentes (si las hay). Opcionalmente, es posible poblar la tabla de datos incluyendo como tercer parámetro el nombre de un array de cadenas (STRING). Si no se dispone de datos aún, basta con no especificar este parámetro.

Como alternativa a añadir los datos cuando se crea cada columna, una tabla puede poblarse mediante la función TABLE_ADDLINE .

Es posible utilizar esta función para crear un array de gráficos en lugar de cadenas. Para ello, basta con indicar como tercer parámetro el nombre de un array de enteros (INT) en lugar de un array de STRING.

VEA TAMBIÉN
TABLE Crea una tabla o lista
TABLE_REMOVECOLUMN Quita una columna de una tabla
